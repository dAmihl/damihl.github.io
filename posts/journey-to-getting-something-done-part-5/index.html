Journey to Getting Something Done - Part 5

March 19, 2020

The virus situation

The Coronavirus also hit my home country Austria in the last couple weeks which is not a particularly surprising thing for a pandemic virus to do. What we usually don't expect such a virus to do is help us making some progress with our games. Of course I'd prefer a no-virus-situation over the current crisis but, well, it is what it is.

Because I am currently not allowed to leave my flat (except for getting groceries of course) I have a lot more time on the evenings. Which resulted in getting my tapping mobile game to a much better and much closer-to-finished state!

Got something done

To be honest I would even say that the current state is finished enough for me to call this experiment a success. I have created a (simple) game; created assets; iterated on the gameplay and created something which is at somewhat interesting.

There is not much that is completely new though. I have changed the theme a little. I was thinking about the hero shooting lasers in a city right? The hero is now a spheric robot shooting lasers out of his eye and cheering to itself!

What I actually added (as I mentioned in earlier posts) is the possibility to add the color scheme of the main mechanic (the lasers and shields). This is now done via a button in the main menu. Have a look:

What I'm doing is storing the current scheme in a global class and using the colors defined there to pass to the fragment shaders for the specific sprites (laser, enemy shields, the robot changing color while charging).

Here's how the game now looks:

The shooting button disappeared and is now done via tapping anywhere on the screen, which on one hand makes more space for the “play area”, and on the other hand frees me from the burden to create a fitting button sprite :)

I refined the shields a little as to make them thicker and thus easier to identify.

All in all I think this was a successful project. I created something from start to finish. I learned a lot using Godot Engine and Aseprite.

And I even released it - at least on itch.io :)

But most importantly: I finally got something done!

Now to one of the next ideas that I got. Let's try to get more things done now, ok?
